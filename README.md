# Los 11 de Sampa

🟢 **EN PRODUCCIÓN** — live en **[los11desampa.com](https://www.los11desampa.com)**

Fantasy football del **Mundial FIFA 2026**, para Argentina/LatAm. El usuario arma un equipo
dentro de un presupuesto, suma puntos por el rendimiento real de los jugadores en cada fecha
del Mundial, y compite en ranking global + ligas privadas.

> ⚠️ La app ya tiene usuarios reales, equipos guardados y pagos en producción. Antes de tocar
> algo que pueda impactar lo live (DB, deploy, scoring, pagos), leé **[`docs/PRODUCCION.md`](./docs/PRODUCCION.md)**.
